
데이터 추가

```python
a = [1, 2, 3]
a.append(4)
a = [1,2,3,4]
```

데이터 할당

```python
a[0] = 0
a = [0,2,3,4]
```

데이터 삭제

```python
del a[3]
a = [0, 2, 3]
```

슬라이싱
- 리스트 [시작:끝+1]

```python
b = [3, 4, 2, 1]
b[1:3]
[4, 2]
```

리스트 길이, 갯수
- len(리스트)

```python
b = [3, 4, 2, 1]
len(b)
4
```

리스트 정렬
- 리스트.sort() 오름차순으로 정렬된다
- 리스트.sort(reverse=True) 내림차

```python
b = [3, 4, 2, 1]
b.sort()
b = [1, 2, 3, 4]
```
